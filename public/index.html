Getting Started

Include Phx.js

First we need to include the phx.js library on the page. The library occupies a global variable of Phx. 


<script src=“phx.js”></script>

Alternatively, if you’re using an AMD loader for JavaScript modules, that is also supported.


// Using requirejs
        require([‘path/to/phx.js’], function (Phx) {
          // Use Phx as normal here.
          ...
        });

You can also grab Phx.js using npm:


npm install phx --save

Creating a chart

To create a chart(s), we need 3 things, a reference to a valid DOM element, a set of options, and data. Phoenix can create one or more charts based on the data that it is passed and the layout type specified.

Passing in a DOM Element

First, we need to instantiate the Phx class. To do this, we need to pass a reference to a valid DOM element. Let's say in our HTML file, we have a <div> where our chart will go. 


<div id=“myChart” style=“width:400px; height:400px”></div>

Using vanilla javascript, we can select the DOM node and pass it into our Phx function.



        var el = document.getElementById(“myChart”);

        // Instantiate our chart.
        var myChart = new Phx(el);

Or, using jQuery.



        var el = $(“#myChart”).get(0);
        var myChart = new Phx(el);

Or, using d3.



        var el = d3.select(“#myChart”)[0][0];
        var myChart = new Phx(el);

Structuring the Data

Since Phx can create multiple charts, data passed to Phx must be in the form of a chart object. So, if you are creating a single chart, then the chart object can be passed directly to the data method. If you are creating more than one chart, then an array of chart objects need to be passed to the data method.

For example



        // Single Chart
        var singleChart = {
          // accessor
          data: [ {x: "apple", y: 10}, {x: "orange", y: 12}, {x: "banana", y: 15} ]
        };

        myChart.data(singleChart);

        // Multiple Charts
        var multipleCharts = [
          { data: [...] }, // Chart 1
          { data: [...] }, // Chart 2
          { data: [...] }, // Chart 3
        ];

        myChart.data(multipleCharts);

Setting the Options

The options is a javascript object of key value pairs. We may pass in options at the global level or at the chart level.

Global options are the default options applied to all charts. 

        // Global options
        var options = {
          // Type of chart, defaults to series chart
          type: "series",
          // Chart layout, applies when there are multiple charts, defaults to rows
          // Options include rows, columns, or grid
          layout: "rows",
          // Where to pull the data from each chart object, defaults to the data key
          accessor: "data",
        };

        // Pass the options object to the options method to set the global options.
        myChart.options(options);
