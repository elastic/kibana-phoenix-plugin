Bar Charts

A bar chart is a chart that uses either horizontal or vertical bars to show comparisons among categories or for time series.

Stacked Time Series Chart

Horizontal Time Series Chart

Example usage

var myBarChart = new Phx(el)
  .data(data)
  .options(options)
  .draw()

Data Structure

{{ {size: 5} | getData}}

Data is passed in through the Chart Object. For time series bar charts, data key references an array of objects for single series or an array of arrays of objects for stacked series.
