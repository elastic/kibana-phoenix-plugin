Getting Started

Include Phx.js

First we need to include the phx.js library on the page. The library occupies the global variable Phx. 

<script src=“phx.js”></script>

Alternatively, you can use an AMD loader for JavaScript modules.

// Using requirejs
require([‘path/to/phx.js’], function (Phx) {
  // Use Phx as normal here.
  ...
});

You can also grab phx.js using npm:

npm install phx --save
